## 📱 使用方式

### 阅读文章
1. 在文章区域上下滑动查看完整内容
2. 每篇文章约5-6段
3. 所有5道题共享同一篇文章

1. 阅读完整文章
2. 查看当前题目（如"21. According to..."）

4. 选项变蓝色表示已选中
5. 答案自动保存

1. 点击"下一题"按钮
2. 或使用题号切换器（仅完形填空1-20题可见）

## 🔧 技术实现

### 数据结构

每道阅读理解题包含：
```java
new ExamQuestion(
    QuestionType.READING_COMPREHENSION,  // 题目类型
    "阅读理解 Text 1 - 第21题",          // 题目标题
    text1Passage,                        // 完整文章内容
    "21. 题目问题",                      // 题目文本
    new String[]{                        // 4个选项
        "选项A", "选项B", "选项C", "选项D"
    },
    2,                                   // 正确答案索引(0-3)
    "答案解析"                           // 解析说明
)
```

### 变量复用

每篇文章使用一个String变量存储完整内容：
- `text1Passage` - Text 1完整文章
- `text2Passage` - Text 2完整文章
- `text3Passage` - Text 3完整文章
- `text4Passage` - Text 4完整文章

5道题共享同一个passage变量，节省代码量。
