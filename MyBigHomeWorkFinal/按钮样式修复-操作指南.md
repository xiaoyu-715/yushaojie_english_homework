- `app/src/main/res/drawable/btn_correct_background.xml` - 浅绿色背景 + 深绿色边框

### 2. ✓ 清理了构建缓存
- 已删除 `app\build` 目录
- 已删除相关资源缓存

## 🚀 接下来您需要做的（3步）

### 第1步：在Android Studio中清理项目

打开Android Studio，依次点击：

```
Build → Clean Project
```

等待清理完成（底部进度条消失）

### 第2步：重新构建项目

继续点击：

```
Build → Rebuild Project
```

等待构建完成（可能需要1-2分钟）

### 第3步：运行应用

按 `Shift + F10` 或点击绿色运行按钮 ▶️

## 🎨 预期效果

运行应用后，进入"词汇训练"页面：

### 初始状态（未回答）

## ❓ 如果还是没有变化？

### 方案A：使用IDE的缓存清理

```
File → Invalidate Caches / Restart... → Invalidate and Restart
```

等待Android Studio重启后，重复上面的第1-3步。

### 方案B：卸载应用重新安装

1. 在设备/模拟器上**卸载**当前应用
2. 在Android Studio中重新点击运行按钮
3. 应用会重新安装并启动

### 方案C：使用批处理脚本

双击运行项目根目录的 `clean_and_rebuild.bat`，然后重复第1-3步。

## 📱 测试步骤

1. 打开应用
2. 点击首页的 **"词汇训练"** 卡片
3. 观察4个选项按钮 → 应该都是**蓝色**
4. 随意选择一个答案
5. 观察颜色变化 → 正确答案变**绿色**，如果选错则错误答案变**红色**
6. 点击 **"下一题"**
7. 新题目的按钮应该重新变回**蓝色**

## ✨ 技术细节

修改前后对比：

| 项目 | 修改前 | 修改后 |
|-----|-------|-------|
| 背景 | 白色渐变 → 灰白色 | 纯色（蓝/绿/红）|
| 边框宽度 | 1dp | 2dp |
| 边框颜色 | 浅色 | 深色（更明显）|

这样按钮会更加醒目，状态区分更清晰！
