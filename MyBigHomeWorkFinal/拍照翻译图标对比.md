# 拍照翻译图标对比

## 📊 新旧图标对比

### 旧图标 (ic_translate)
```
┌─────────────┐
│      🔄     │  通用翻译图标
│     ↔️      │  仅表示翻译功能
│             │  不够直观
└─────────────┘
```
**问题**:
- ❌ 只表示翻译，未体现"拍照"
- ❌ 功能表达不够准确
- ❌ 用户需要阅读文字才能理解

---

### 新图标 (ic_camera_translate)
```
┌─────────────┐
│          ⭕ │  翻译徽章 (橙色)
│     📷      │  相机主体 (蓝色)
│    ⚪️     │  相机镜头
└─────────────┘
```
**优势**:
- ✅ 清晰表达"拍照 + 翻译"
- ✅ 视觉层次分明
- ✅ 配色醒目协调
- ✅ 功能一目了然

---

## 🎯 设计元素详解

### 1. 相机主体 (蓝色)
- 代表"拍照"功能
- 使用应用主题色
- 占据主要视觉空间

### 2. 相机镜头 (白色)
- 增加真实感
- 提供视觉焦点
- 双层结构更立体

### 3. 翻译徽章 (橙色)
- 代表"翻译"功能
- 醒目的对比色
- 右上角位置显眼

### 4. 译字符号 (白色)
- 简化的"译"字
- 白色保证可读性
- 强化翻译概念

## 📐 技术规格对比

### 旧图标
- 格式: Vector Drawable
- 颜色: 1种 (白色)
- 元素: 2-3个path
- 复杂度: 简单

### 新图标
- 格式: Vector Drawable
- 颜色: 3种 (蓝/橙/白)
- 元素: 4个path
- 复杂度: 适中
- 文件大小: 约1.2KB
